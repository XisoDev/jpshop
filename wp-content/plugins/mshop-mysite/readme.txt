엠샵 마이 사이트 플러그인은 웹사이트 소유권 처리 및 광고 전환 추적을 쉽게 할 수 있도록 도와주는 플러그인

== Description ==

[Korean]

* 간단하고 쉽게 웹사이트 소유권 인증
* 구글, 페이스북 전환 추적을 통한 광고 효과 측정


<간단하고 쉽게 웹사이트 소유권 인증>

구글, 네이버, 야후, 빙 등 검색 사이트 등록을 위해서는 기본적으로 웹사이트 소유권 인증이 되어야 합니다.

그러나 HTML 에 익숙하지 않은 경우, 웹사이트 소유권 인증 및 확인은 어려운 일이죠.

엠샵 마이 사이트 플러그인은 HTML 지식이 없는 일반인도 웹사이트 소유권 인증을 쉽게 할 수 있도록 도와주는 플러그인 입니다.

내 사이트를 등록할 검색 매체의 웹마스터도구에서 발급 된 메타 태그를 복사하여 입력하는 것으로 웹사이트 소유권 확인을 할 수 있습니다.


<네이버, 구글, 페이스북 광고 효과 측정>

구글, 페이스북, 네이버에 광고 진행을 위한 시작 후, 광고 효과 측정을 위해서는 분석 스크립트를 설치해야 합니다.

그러나 워드프레스 구조에 익숙하지 않는 관리자는 스크립트 삽입에 어려움을 호소합니다.

엠샵 마이 사이트 플러그인은 광고 매체에서 제공하는 아이디를 입력하는 것으로 광고 효과 측정을 쉽게 시작할 수 있습니다.

쇼핑몰, 홈페이지 등 웹사이트는 검색 사이트에 검색이 되어야 하며, 광고 진행 시 광고 효과 분석도 진행 하여야 합니다.

검색 사이트에 내 사이트는 왜 나오지 않지? 광고 분석의 시작을 고민하지 마시고, 지금 바로 엠샵 마이 사이트 플러그인으로 웹사이트 관리를 시작 해 보세요.

[English]

* Simple & Easy Website verify Ownership
* Naver, Google, Facebook advertisement effect measurement


<Simple & Easy Website verify Ownership>

To register your site to search engine, such as Google, Naver, Yahoo, or Bing, website ownership must be authenticated.

However, if you are not familiar with HTML, website ownership authentication and verification is difficult.

The M-Shop My Site plug-in makes it easy for anyone without HTML knowledge to authenticate website ownership.

You can verify website ownership by copying and entering the meta tag issued by webmaster tools on the search engine where you want to register your site.


<Naver, Google, Facebook advertisement effect measurement>

After starting to advertise on Google, Facebook, and Naver you need to install an analysis script to measure your advertising effectiveness.

However, administrators who are unfamiliar with the wordpress structure are having trouble embedding scripts.

The M-Shop My Site plug-in allows you to easily start measuring your advertising effectiveness by entering the ID provided by the advertising medium.

Web sites such as shopping malls and homepages should be searched on the search site, and the advertisement effect should be analyzed when the advertisement is performed.

Why does not my site appear on search sites? Do not worry about starting the analysis of your ads, and start managing your website with the M-Shop My Site plug-in today.

== Installation ==

[Korean]

1. 워드프레스 플러그인 화면에서 MShop Mysite 검색하여 직접 설치하거나, 다운로드 받은 파일을 /wp-content/plugins/mshop-mysite 디렉토리에 업로드하여 설치 합니다.
2. MShop Mysite 플러그인을 활성화합니다.
3. MShop Mysite 메뉴에서 플러그인 설정을 진행 합니다.
4. 등록할 웹사이트 매체를 선택 후, 매체에서 발급 받은 HTML 메타태그를 입력하고 저장합니다.
5. 광고를 진행하는 경우 전환 추적 탭에서 광고 매체를 선택 한 후 전달 받은 아이디를 입력 후 저장 합니다.


<웹사이트 소유권 HTML 발급하기>

구글 : 구글 웹마스터도구 (https://www.google.com/webmasters)
네이버 : 네이버 웹마스터도구 (http://webmastertool.naver.com)


[English]

1. Upload the plugin files to the /wp-content/plugins/mshop-mysite directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'MShop MySite' screen in WordPress
3. Use the 'MShop MySite' menu screen to configure the plugin.
4. After then, check your service site owner verification or website source in section.
5. If you're running an ad, select the ad medium on the Conversion Tracking tab, enter the delivery ID and save it.


<Get Website ownership HTML>

Google Webmaster : https://www.google.com/webmasters
Naver Webmaster : http://webmastertool.naver.com
